Clement

Homepage: http://www.clement.safe.ca/
Supports: libclamav
Clement is an application working as an mail server firewall. Its only purpose is to trap most of the unwanted mail at the earliest possible stage. Clement uses ClamAV to detect viruses and SpamAssassin to check for spam content.
