Error handling

Use cl_strerror() to convert error codes into human readable messages. The function returns a statically allocated string: 
	if(ret != CL_SUCCESS) {
	    printf("cl_load() error: %s\n", cl_strerror(ret));
	    cl_engine_free(engine);
	    return 1;
	}
