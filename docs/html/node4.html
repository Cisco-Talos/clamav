Mailing lists

There are four mailing lists available:
  • clamav-announce*lists.sf.net - info about new versions
    (including debian package releases), moderated1.
  • clamav-users*lists.sf.net - user questions
  • clamav-devel*lists.sf.net - developement
  • clamav-virusdb*lists.sf.net - database update announcements
You can subscribe and check the mailing list archives at: http://www.clamav.net/ml.html
