Freshclam

freshclam is ClamAV's virus database update tool and reads it's configuration from the file 'freshclam.conf' (this may be overriden by command line options). Freshclam's default behavior is to attempt to update databases that are paired with downloaded cdiffs. Potentially corrupted databases are not updated and are automatically fully replaced after several failed attempts unless otherwise specified.

Here is a sample usage including cdiffs: 
$ freshclam
